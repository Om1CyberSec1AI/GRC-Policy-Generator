# GRC Policy Generator

Generate customizable policies for GRC, aligned with NIST CSF and PCI DSS.

## Features
- Web interface for input.
- Outputs policies in Markdown or DOCX.
- Covers key areas like risk management and compliance.

## Installation
1. Clone: `git clone https://github.com/yourusername/GRC-Policy-Generator.git`
2. `pip install -r requirements.txt`
3. Run: `flask run`

## Usage
- Visit http://127.0.0.1:5000
- Fill form and download policy.

from flask import Flask, render_template, request, send_file
from docx import Document

app = Flask(__name__)

@app.route('/', methods=['GET', 'POST'])
def index():
    if request.method == 'POST':
        org_name = request.form['org_name']
        # Generate DOCX
        doc = Document()
        doc.add_heading(f"{org_name} GRC Policy", 0)
        doc.add_paragraph("Aligned with NIST CSF: Identify risks...")
        doc.save('policy.docx')
        return send_file('policy.docx', as_attachment=True)
    return render_template('index.html')

if __name__ == '__main__':
    app.run(debug=True)
